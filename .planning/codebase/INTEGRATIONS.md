## CDN & Assets

**Font Loading:**
- Google Fonts loaded via `<link>` tag in `src/layouts/Layout.astro`
- Fonts: Bricolage Grotesque (headings, 700/800), DM Sans (body, 400/500/700), Fira Code (mono, 400)
- URL: `https://fonts.googleapis.com/css2?family=Bricolage+Grotesque:wght@700;800&family=DM+Sans:wght@400;500;700&family=Fira+Code:wght@400&display=swap`
- Preconnect hints to `fonts.googleapis.com` and `fonts.gstatic.com`
- Font fallback metrics defined in `src/styles/global.css` to reduce CLS (Cumulative Layout Shift)

**OG Image Fonts (build-time only):**
- Inter Regular (`src/assets/fonts/Inter-Regular.woff`) — used by `satori` for OG image generation
- Space Grotesk Bold (`src/assets/fonts/SpaceGrotesk-Bold.woff`) — used by `satori` for OG image generation
- These are loaded from disk at build time, not served to browsers

**Image Optimization:**
- Astro `<Image>` component used for all images (auto-optimization via `sharp`)
- Source images stored in `src/assets/images/` (JPG, PNG, JPEG)
- Hero image loaded with `loading="eager"` and `fetchpriority="high"`
- OG images dynamically generated at build time via `satori` + `sharp` pipeline (`src/lib/og-image.ts`, `src/pages/open-graph/[...slug].png.ts`)

**Static Asset Hosting:**
- GitHub Pages serves all static assets
- No external CDN configured beyond Google Fonts

## Third-Party Libraries (Runtime)

**Client-side JavaScript libraries loaded at runtime:**

| Library | Load Method | Purpose |
|---------|------------|---------|
| GSAP + ScrollTrigger | Bundled via npm import | Scroll-triggered animations, parallax, batch reveals |
| Lenis | Bundled via npm import | Smooth scroll (desktop only) |
| vanilla-tilt | Dynamic import (`await import('vanilla-tilt')`) | 3D tilt hover effect on cards (desktop only) |

**No CDN-loaded runtime libraries.** All JS dependencies are bundled by Vite/Astro at build time.

**Runtime behavior notes:**
- All animation libraries respect `prefers-reduced-motion: reduce` — animations are disabled or simplified
- Touch device detection (`pointer: coarse`) disables smooth scroll, tilt, and custom cursor
- GSAP ScrollTrigger instances and Lenis are cleaned up on Astro view transitions (`astro:before-swap`)
