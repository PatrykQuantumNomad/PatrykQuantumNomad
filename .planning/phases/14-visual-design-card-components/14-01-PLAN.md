```css
.card-hover:hover {
  transform: translateY(-4px);
  box-shadow:
    0 12px 40px var(--category-glow, var(--color-accent-glow)),
    0 0 60px var(--category-glow, var(--color-glow-neon));
  border-color: var(--color-accent);
}
```

Also add `animate-pulse` disable to the existing `@media (prefers-reduced-motion: reduce)` block (around line 509):
```css
.animate-pulse {
  animation: none;
```
