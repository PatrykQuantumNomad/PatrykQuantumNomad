  <name>Task 2: Create InteractiveGraph.tsx React component with pan/zoom and zoom reset</name>
  <files>src/components/ai-landscape/InteractiveGraph.tsx</files>
  <action>
    Create `src/components/ai-landscape/InteractiveGraph.tsx` -- a React island component that renders the AI landscape graph as interactive SVG with d3-zoom.

    **Imports:**
    - `useRef, useEffect, useState, useCallback, useMemo` from 'react'
    - `select` from 'd3-selection'
    - `zoom, zoomIdentity, type ZoomTransform` from 'd3-zoom'
    - `GraphProps, ROOT_NODE_RADIUS, NODE_RADIUS, LABEL_FONT_SIZE, FONT_FAMILY, stripParenthetical` from '../../lib/ai-landscape/graph-data'

    **Component signature:**
    ```typescript
    export default function InteractiveGraph({ nodes, edges, positions, clusters, meta }: GraphProps)
    ```

    **Refs:**
    - `svgRef = useRef<SVGSVGElement>(null)` -- for d3-zoom attachment
    - `zoomRef = useRef<ReturnType<typeof zoom<SVGSVGElement, unknown>>>()` -- for reset button access
    - `containerRef = useRef<HTMLDivElement>(null)` -- outer wrapper for tooltip positioning (Plan 02)

    **State:**
    - `transform: ZoomTransform` (initialized to `zoomIdentity`) -- current pan/zoom transform
    - `showZoomHint: boolean` (false) -- controls modifier key hint overlay visibility
    - `hoveredNode: string | null` (null) -- ID of hovered node (for Plan 02 tooltip, stub state now)
    - `hoveredEdge: string | null` (null) -- key of hovered edge (for Plan 02 edge labels, stub state now)

    **Memoized lookups:**
    - `posMap = useMemo(() => new Map(positions.map(p => [p.id, p])), [positions])`
    - `nodeMap = useMemo(() => new Map(nodes.map(n => [n.id, n])), [nodes])`
    - `clusterMap = useMemo(() => new Map(clusters.map(c => [c.id, c])), [clusters])`
    - `rootNodeIds = useMemo(() => new Set(nodes.filter(n => n.parentId === null).map(n => n.id)), [nodes])`

    **d3-zoom setup (useEffect, empty deps):**
    ```typescript
    useEffect(() => {
      const svg = svgRef.current;
      if (!svg) return;

      const zoomBehavior = zoom<SVGSVGElement, unknown>()
        .scaleExtent([0.3, 4])
        .filter((event: Event) => {
          // Allow drag, touch events unconditionally
          if (event.type !== 'wheel') return true;
          // Wheel events require Ctrl (Win/Linux) or Meta/Cmd (Mac)
          return (event as WheelEvent).ctrlKey || (event as WheelEvent).metaKey;
        })
        .on('zoom', (event) => {
          setTransform(event.transform);
        });

      zoomRef.current = zoomBehavior;
      select(svg).call(zoomBehavior);

      // Show hint overlay on unmodified wheel
      const handleWheel = (e: WheelEvent) => {
        if (!e.ctrlKey && !e.metaKey) {
          setShowZoomHint(true);
        }
      };
      svg.addEventListener('wheel', handleWheel, { passive: true });

      return () => {
        select(svg).on('.zoom', null);
        svg.removeEventListener('wheel', handleWheel);
      };
    }, []);
    ```

    **Zoom hint auto-hide (useEffect on showZoomHint):**
    When `showZoomHint` becomes true, set a 2-second timeout to hide it. Clear timeout on cleanup.

    **resetZoom callback:**
    ```typescript
    const resetZoom = useCallback(() => {
      const svg = svgRef.current;
      const zoomBehavior = zoomRef.current;
      if (!svg || !zoomBehavior) return;
      select(svg).transition().duration(500).call(zoomBehavior.transform, zoomIdentity);
    }, []);
    ```

    **CSS styles block (inside SVG, matching svg-builder.ts):**
    Build a `<style>` element inside the SVG with:
    - Cluster fill/stroke rules: `.ai-cluster-{id} { fill: color; stroke: darkColor; stroke-width: 1.5; }`
    - Dark mode overrides: `html.dark .ai-cluster-{id} { fill: darkColor; stroke: color; }`
    - Edge style: `.ai-edge { stroke: var(--color-border); }`
    - Label style: `.ai-label { fill: var(--color-text-primary); font-family: 'DM Sans', sans-serif; font-size: 9px; pointer-events: none; }`
    - Edge label style: `.ai-edge-label { fill: var(--color-text-secondary); font-family: 'DM Sans', sans-serif; font-size: 8px; pointer-events: none; }`

    Use `useMemo` to build the CSS string from the clusters array to avoid rebuilding on every render.

    **Render structure:**
    ```tsx
    <div ref={containerRef} className="relative">
      <svg
        ref={svgRef}
        viewBox={`0 0 ${meta.width} ${meta.height}`}
        className="w-full h-auto"
        style={{ maxWidth: `${meta.width}px`, cursor: 'grab' }}
        role="img"
        aria-label={`Interactive AI Landscape graph with ${nodes.length} concepts across ${clusters.length} clusters`}
      >
        <style dangerouslySetInnerHTML={{ __html: cssString }} />
        <g transform={transform.toString()}>
          {/* Edges layer (rendered first for z-order) */}
          <g className="edges">
            {edges.map((edge) => {
              const src = posMap.get(edge.source);
              const tgt = posMap.get(edge.target);
              if (!src || !tgt) return null;

              let strokeWidth = '1';
              let strokeDasharray: string | undefined;
              let opacity: number | undefined;

              if (edge.type === 'hierarchy') {
                strokeWidth = '2';
              } else if (edge.type === 'includes') {
                strokeWidth = '1.5';
                strokeDasharray = '4 3';
              } else {
                opacity = 0.4;
              }

              const edgeKey = `${edge.source}-${edge.target}`;
              return (
                <g key={edgeKey}>
                  <line
                    x1={src.x} y1={src.y} x2={tgt.x} y2={tgt.y}
                    className="ai-edge"
                    strokeWidth={strokeWidth}
                    strokeDasharray={strokeDasharray}
                    opacity={opacity}
                  />
                </g>
              );
            })}
          </g>

          {/* Nodes layer */}
          <g className="nodes">
            {nodes.map((node) => {
              const pos = posMap.get(node.id);
              if (!pos) return null;
              const r = rootNodeIds.has(node.id) ? ROOT_NODE_RADIUS : NODE_RADIUS;
              const cluster = clusterMap.get(node.cluster);
              const clusterClass = cluster ? `ai-cluster-${cluster.id}` : '';
              const label = stripParenthetical(node.name);
              const labelY = pos.y + r + LABEL_FONT_SIZE + 2;

              return (
                <g key={node.id}>
                  <circle cx={pos.x} cy={pos.y} r={r} className={clusterClass} />
                  <text x={pos.x} y={labelY} textAnchor="middle" className="ai-label">
                    {label}
                  </text>
                </g>
              );
            })}
          </g>
        </g>
      </svg>

      {/* Zoom-to-fit reset button */}
      <button
        onClick={resetZoom}
        className="absolute top-3 right-3 bg-[var(--color-surface)] border border-[var(--color-border)] rounded-lg px-3 py-1.5 text-xs font-mono text-[var(--color-text-secondary)] hover:text-[var(--color-text-primary)] hover:border-[var(--color-text-secondary)] transition-colors shadow-sm"
        title="Reset zoom to fit all nodes"
        aria-label="Reset zoom to fit all nodes"
      >
        Reset View
      </button>

      {/* Modifier key zoom hint overlay */}
      {showZoomHint && (
        <div className="absolute inset-0 flex items-center justify-center pointer-events-none">
          <div className="bg-black/70 text-white text-sm px-4 py-2 rounded-lg">
            Use <kbd className="font-mono bg-white/20 px-1.5 py-0.5 rounded text-xs">Ctrl</kbd> + scroll to zoom
          </div>
        </div>
      )}
    </div>
    ```

    **Critical implementation notes:**
    - The transform is applied to the single wrapping `<g>` element, NOT to individual nodes. This means React only diffs one attribute change per zoom frame, not 117+ elements. (Research Pitfall 2)
    - The `<style>` block is inside the SVG using `dangerouslySetInnerHTML` because React doesn't support `set:html` (that's Astro-only). This ensures cluster CSS classes and dark mode overrides work inside the React-rendered SVG. (Research Pitfall 6)
    - d3-zoom is attached to the `<svg>` element (not the inner `<g>`) so it captures all mouse/touch events across the full viewport. (Research anti-pattern)
    - The component uses `cursor: 'grab'` on the SVG for visual affordance of drag-to-pan.
    - Do NOT import d3-transition explicitly -- d3-zoom depends on it internally. (Research anti-pattern)
    - Use `className` (not `class`) -- this is React JSX, not Astro.

    IMPORTANT: All imports must be relative paths (`../../lib/ai-landscape/graph-data`).
    IMPORTANT: Export as default function (required for client:only="react" islands).
  </action>
