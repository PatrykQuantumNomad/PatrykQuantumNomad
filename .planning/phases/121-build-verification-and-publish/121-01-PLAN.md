- `--output=json`
- `--chrome-flags="--headless=new"` (eliminate rendering overhead)
- `--quiet` (reduce noise)

Run 3 separate invocations, saving each to a different output file (e.g., `$TMPDIR/lh-run-1.json`, `$TMPDIR/lh-run-2.json`, `$TMPDIR/lh-run-3.json`).

- Performance: `categories.performance.score * 100`
- Accessibility: `categories.accessibility.score * 100`
- Best Practices: `categories['best-practices'].score * 100`
- SEO: `categories.seo.score * 100`

Compute the **median** of each category across the 3 runs. Report all 3 runs plus the median.

**Step 4 -- Evaluate against threshold**

Pass criteria: All 4 median category scores are 90 or above.

If any category median is below 90, report the specific category, its scores across runs, and the median. Do NOT attempt to fix anything -- this is a verification phase. Flag the failure for investigation.

**Step 5 -- Clean up**

Kill the preview server process. Remove temporary Lighthouse JSON files from TMPDIR.
