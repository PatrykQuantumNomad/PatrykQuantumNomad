    - Compute `currentIndex = chapters.findIndex(ch => ch.slug === currentSlug)`
    - Compute `prev = currentIndex > 0 ? chapters[currentIndex - 1] : null`
    - Compute `next = currentIndex < chapters.length - 1 ? chapters[currentIndex + 1] : null`
    - Render `<nav aria-label="Chapter navigation" class="mt-12 pt-8 border-t border-[var(--color-border)]">`

    - Use `text-[var(--color-accent)]` for link text, `text-[var(--color-text-secondary)]` for labels
    - If prev is null, render empty `<div />`. Same for next.
