<objective>
Generate and commit 7 standard Jupyter Notebook .ipynb files to the repository so Google Colab can open them via the GitHub URL scheme.

Purpose: Colab's `colab.research.google.com/github/{user}/{repo}/blob/{branch}/{path}` pattern requires the .ipynb files to be committed to the git repo. Without committed files, the "Open in Colab" badges from Plan 02 would produce 404 errors.

Output: 7 .ipynb files at `notebooks/eda/{slug}.ipynb`, a reusable generation script at `scripts/generate-notebooks.ts`, and validation tests confirming file integrity.
</objective>
