       - diagramSvgOpen(config, ariaLabel) -- root SVG tag with viewBox, role="img", aria-label, responsive width

       - arrowLine(x1, y1, x2, y2, markerId) -- line with marker-end reference
       - arrowMarkerDef(id) -- SVG marker definition for directional arrows
       - textLabel(x, y, text, opts) -- centered text with configurable font-size, fill, weight

       - Layout: 8 horizontally-centered rounded rectangles stacked vertically with arrow lines between them

       - Layer order (outermost to innermost): Trusted Host, CORS, Security Headers, Rate Limiting, Request ID, Logging, Prometheus Metrics, Auth
