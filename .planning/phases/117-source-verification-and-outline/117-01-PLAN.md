**Step 1: Create URL verification list**

**Step 2: Batch-verify URLs**

Run HTTP HEAD requests for all URLs using curl:
```bash
while IFS='|' read -r num url; do
  status=$(curl -sI -o /dev/null -w "%{http_code}" -L --max-time 10 "$url")
  echo "$num|$status|$url"
done < sources-urls.txt
```

Interpret results:
- 200: VERIFIED
- 301/302: VERIFIED (redirect -- note final destination)
- 403: FLAGGED (likely login wall -- check if ResearchGate or paywalled)
- 404: UNREACHABLE
- 000: UNREACHABLE (timeout/DNS failure)
