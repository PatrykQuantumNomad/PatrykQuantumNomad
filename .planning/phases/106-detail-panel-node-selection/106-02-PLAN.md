**BottomSheet.tsx — create mobile bottom sheet wrapper:**

A wrapper component for the DetailPanel content on mobile (< 768px).

Props:
```ts
interface BottomSheetProps {
  isOpen: boolean;
  onClose: () => void;
  children: React.ReactNode;
}
```

Rendered structure:
1. A semi-transparent backdrop overlay (`fixed inset-0 bg-black/30 z-40`) that calls onClose on click
2. A sheet container (`fixed bottom-0 left-0 right-0 z-50 bg-[var(--color-surface)] border-t border-[var(--color-border)] rounded-t-2xl`) with:
   - A drag handle bar at top (small centered gray pill, 40px wide, 4px tall, rounded)
   - Content area with `max-h-[60vh] overflow-y-auto p-4` that renders `{children}`
3. Slide-up animation: use CSS `transform: translateY(100%)` when closed, `translateY(0)` when open, with `transition-transform duration-300 ease-out`
4. When `isOpen` is false, do not render (return null) — or use visibility+transform for animation. Simplest approach: always render, use conditional transform + pointer-events-none when closed, but only render children when open to avoid waste. Best approach: render the container with transform animation, but conditionally render children only when open.

Actually, simplest correct approach: if `!isOpen`, return null. The 300ms slide-in animation on mount can be achieved via a useEffect that sets a `visible` state after mount:
- Render the component when isOpen is true
- On mount, set `isVisible = false`, then after a requestAnimationFrame, set `isVisible = true` to trigger the CSS transition from translateY(100%) to translateY(0)
- This gives a smooth slide-up on open

Add ancestry highlight handler:
```ts
const handleShowAncestry = useCallback((nodeSlug: string) => {
  // Build a Map<string, AiNode> from the slug for ancestry lookup
  const slugMap = new Map(nodes.map(n => [n.slug, n]));
  const chain = buildAncestryChain(nodeSlug, slugMap);
  const chainSlugs = chain.map(a => a.slug);
  // Also include the node itself in the highlight
  const node = slugMap.get(nodeSlug);
  const highlightIds = new Set<string>();
  for (const n of nodes) {
    if (chainSlugs.includes(n.slug) || n.slug === nodeSlug) {
      highlightIds.add(n.id);
    }
  }
  setHighlightedNodeIds(highlightIds);
}, [nodes]);
```

Add close handler:
