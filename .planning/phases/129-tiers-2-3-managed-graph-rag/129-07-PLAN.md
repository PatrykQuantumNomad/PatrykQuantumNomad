4. **Expected cost** (verbatim from 129-RESEARCH.md "Cost estimates"):
   | Operation | Calls | Avg tokens | Cost |
   |-----------|-------|-----------|------|
   | One-time entity extraction (~1 LLM call per chunk, ~420 chunks) | 420 | 3K in + 600 out | **~$1.00** |
   | One-time embedding (~420 chunks × ~1.2K tokens) | — | 500,000 | ~$0.01 |
   | Per-query LLM (graph traversal + answer) | 1-3 | 5,400 | ~$0.005 |
   | Per-query embedding (keyword extraction) | 1 | 30 | <$0.000001 |

   **Comparison:** Tier 1 ingest ≈ $0.011; Tier 3 ingest ≈ **90× more** ($1.00). The trade is: per-query reasoning quality on multi-hop questions Tier 1 misses entirely.

   - `kv_store_full_docs.json`, `kv_store_text_chunks.json`, `kv_store_doc_status.json`, `kv_store_llm_response_cache.json` — KV stores

6. **Known weaknesses (deliberate)**:
   - **Cost asymmetry vs Tier 1.** $1 vs $0.01 ingest. Justified ONLY when multi-hop reasoning matters.
   - **LightRAG version churn (Pitfall 9).** Pinned EXACTLY at `1.4.15`. Bumping requires re-running `scripts/probe_lightrag_token_tracker.py` to confirm the cost-adapter shape still matches.
   - **`mix` mode may degrade without a reranker (Open Q4).** We default to `hybrid`. If you install `bge-reranker-v2-m3`, switch to `--mode mix` for marginal recall gains.
   - **No image/figure handling.** LightRAG is text-only (default). Tier 4 (Phase 130) covers multimodal via RAG-Anything.
   - **Per-chunk citation surfacing is weaker than Tier 2's.** LightRAG returns synthesized prose; the chunks-table in `render_query_result` is rendered EMPTY for Tier 3 (we render answer-only). Phase 131 evaluation may extract citations from the answer text via heuristics.

   > "Comparing Lewis et al. 2020's RAG and Karpukhin et al. 2020's DPR, how does dense passage retrieval relate to RAG's non-parametric memory?"
   This question requires synthesizing two papers via cross-document entity edges. Tier 1's vector-only top-k often retrieves only one of the two papers. Tier 3's hybrid mode traverses entities (DPR → RAG via shared "non-parametric memory" entity) and surfaces both. Phase 131 will quantify the delta.

8. **Architecture map**:
   ```
   PDFs -> tier-3-graph/ingest.py:extract_full_text (PyMuPDF, Tier-3-owned per Pitfall 11)

        -> shared.display.render_query_result (chunks=[]; answer-only is honest for graph RAG)
   ```

Live e2e test contract (`tests/test_main_live.py`):

```python
"""Tier 3 live e2e — 2-paper subset to keep cost ~$0.05-0.15."""
import asyncio
import os
from pathlib import Path

import pytest

SUBSET_PAPERS = 2  # Cost-bound: ~$0.05-0.15 vs full-corpus $1


@pytest.mark.live
def test_tier3_end_to_end(tier3_live_keys_ok, tmp_path, monkeypatch):
    """End-to-end: 2-paper ingest + graphml exists + hybrid query + cost > 0."""
    # Override WORKING_DIR via monkeypatch so we don't pollute the real graph.
    test_working_dir = tmp_path / "lightrag_test"
    monkeypatch.setattr("tier_3_graph.rag.WORKING_DIR", str(test_working_dir))

    from shared.config import get_settings
    settings = get_settings()
    os.environ["OPENROUTER_API_KEY"] = settings.openrouter_api_key.get_secret_value()

    from shared.cost_tracker import CostTracker
    from shared.loader import DatasetLoader
    from tier_3_graph.rag import build_rag, DEFAULT_LLM_MODEL
    from tier_3_graph.cost_adapter import CostAdapter
    from tier_3_graph.ingest import ingest_corpus
    from tier_3_graph.query import run_query

    # Pick the 2 smallest papers for speed.
    loader = DatasetLoader()
    papers = loader.papers()
    dataset_root = Path(get_settings().dataset_root)
    candidates = [
        (p, (dataset_root / "papers" / p["filename"]).stat().st_size)
        for p in papers
        if (dataset_root / "papers" / p["filename"]).exists()
    ]
    candidates.sort(key=lambda x: x[1])
    subset = [p for p, _ in candidates[:SUBSET_PAPERS]]
    assert len(subset) == SUBSET_PAPERS

    tracker = CostTracker("tier-3-test")
    adapter = CostAdapter(tracker, model=DEFAULT_LLM_MODEL)

    async def _run():
        rag = build_rag(working_dir=str(test_working_dir),
                        llm_token_tracker=adapter,
                        model=DEFAULT_LLM_MODEL)
        await rag.initialize_storages()
        try:
            await ingest_corpus(
                rag, subset,
                console=type("_C", (), {"print": lambda self, *a, **kw: None})(),
                tracker=tracker, model=DEFAULT_LLM_MODEL,
            )
            graphml = test_working_dir / "graph_chunk_entity_relation.graphml"
            assert graphml.exists(), "Graph file not created after ingest"
            answer, in_tok, out_tok = await run_query(
                rag, "What is the main contribution of these documents?",
                mode="hybrid", model=DEFAULT_LLM_MODEL, tracker=tracker,
            )
            assert isinstance(answer, str) and len(answer) > 0, "Empty answer"
            assert tracker.total_usd() > 0, "Cost tracker should record non-zero cost"
        finally:
            await rag.finalize_storages()

    asyncio.run(_run())
```
