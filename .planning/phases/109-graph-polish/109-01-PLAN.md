**1. Import getClusterBounds:**

**2. Memoize cluster bounds:**
Add a `useMemo` that pre-computes a `Map<string, ClusterBounds>` for all clusters:
```typescript
const clusterBoundsMap = useMemo(() => {
  const map = new Map<string, ClusterBounds>();
  for (const c of clusters) {
    const bounds = getClusterBounds(c.id, nodes, posMap);
    if (bounds) map.set(c.id, bounds);
  }
  return map;
}, [clusters, nodes, posMap]);
```

**3. Add zoomToCluster callback:**
Create a `useCallback` named `zoomToCluster` that:
- Takes `clusterId: string`
- Looks up bounds from `clusterBoundsMap`
- Computes scale as `Math.min(meta.width / (x1 - x0), meta.height / (y1 - y0))`
- **Clamp scale to [0.5, 3]** (tighter than zoom extent [0.3, 4]) to prevent extreme zoom
- Builds transform using `zoomIdentity.translate(tx, ty).scale(clampedScale)` where tx/ty center the cluster's bounding box (use `meta.width / 2` and `meta.height / 2` as center point — NOT getBoundingClientRect, consistent with existing zoomToNode)
- Animates via `select(svg).transition().duration(750).call(zoomBehavior.transform, clampedTransform)`
- Depends on: `[clusterBoundsMap, meta.width, meta.height]`

**4. Add interactive cluster legend inside the React island:**
Add a compact horizontal legend strip below the tour selector / search bar area, INSIDE the `<div ref={containerRef}>` but ABOVE the `<div className="flex">`. Place it after the compare mode hints and before the flex container.

Render as a flex wrap row of small clickable cluster labels:
