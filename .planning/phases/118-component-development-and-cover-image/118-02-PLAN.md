**Visual concept -- "Corrupted/Fading Code":**
The motif represents code drifting beyond comprehension. Monospace code-like text fragments are scattered across the dark background at varying opacities -- some nearly legible, some barely visible, some dissolving into the background. This visualizes the essay's central metaphor: code that exists but no one understands.

**Required structure (follow existing cover SVG conventions):**

1. **Defs block** containing:
   - Radial gradient for dark background (`#1a1a2e` center fading to `#0f0f23` edges)
   - Linear gradient for amber accent (`#D97706` to `#F59E0B`)
   - Gaussian blur filter for glow effect on the title
   - Optional noise filter (feTurbulence) for subtle texture

2. **Background:** Full-bleed rect with the radial gradient

3. **Decorative code fragments:** Multiple text elements in the monospace font stack ('SF Mono','Fira Code','Cascadia Code','JetBrains Mono',monospace) scattered across the canvas at various positions. Use code-like snippets that evoke programming without being specific to any language (e.g., function(), return null, // TODO, catch (e), if (unknown), deprecated, void). Each fragment should have a different fill-opacity ranging from 0.03 (nearly invisible) to 0.15 (faintly legible) -- the point is that the code is DARK, barely perceptible. Use small font sizes (10-14px). Rotate some fragments slightly with transform="rotate()" for organic feel. Place 15-25 fragments across the canvas.

4. **Title text:** "DARK CODE" centered at approximately y=290-310, using Helvetica,Arial,sans-serif, font-size around 52-60, fill="white" with fill-opacity="0.95", letter-spacing="12-16", font-weight="bold", text-anchor="middle" at x=600. Apply the amber glow filter to this text. The title should be the brightest, most legible element -- contrasting sharply with the fading code fragments around it.

5. **Subtitle:** Below the title (y approximately 345-355), "The Silent Rot AI Accelerated" in Georgia,'Times New Roman',serif, font-size 16-18, fill="#D97706" with fill-opacity="0.8", italic, centered.

6. **Attribution:** Near bottom (y approximately 585-600), "patrykgolabek.dev/blog/dark-code" in Helvetica,Arial,sans-serif, font-size 11-12, fill="white" with fill-opacity="0.4", centered.

7. **Decorative corner accents:** Four corner brackets using amber lines (#D97706, stroke-opacity 0.3) matching the pattern from claude-code-guide-refresh-cover.svg. These provide a framing device consistent with other covers.

**Key rules:**
- Use ONLY system-safe font stacks (web fonts do not load in img-rendered SVGs)
- Keep the overall mood very dark -- the code fragments should be barely visible
- The title "DARK CODE" should be the clear focal point with everything else receding
- File must be valid XML with the XML declaration
- No external references (no xlink:href to other files, no CSS imports)
