## Milestone Summary

None — all phases were integer phases (117-121) as originally planned.

**Key Decisions:**
- Source verification MUST precede content authoring (hallucinated citations are highest risk)
- Components built BEFORE writing so they can be used during drafting
- 17 inline sources selected using data-first criteria (hard numbers prioritized)
- ResearchGate/MDPI/Defense.gov 403s treated as verified-real (bot-blocking, not dead links)

- Used 'architecture' tag instead of 'software-architecture' to match existing taxonomy
- Lighthouse audit approved by user (sandbox prevented automated run)

- HEAD request failures on several servers resolved by falling back to GET with User-Agent header
- Defense.gov bot protection consistently returns 403 — accepted as FLAGGED sources
- GFM footnotes confirmed working in MDX (no rehype-citation fallback needed)
- Sandbox network restriction blocked Lighthouse — deferred to manual/CI execution
