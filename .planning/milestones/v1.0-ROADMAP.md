## Milestone Summary

**Decimal Phases:**
- None — all phases were integer (no urgent insertions needed)

**Key Decisions:**
- Fresh Astro scaffold over Astro Nano — full creative control for Quantum Explorer theme
- Tailwind CSS for styling — industry standard utility-first CSS
- Class-based dark mode (darkMode: 'class') over media queries for manual toggle support
- CSS custom properties for theme-aware colors — runtime switching without recompilation
- Inline theme-detection script in head — prevents FOUC before first paint
- Astro 5 content collections with glob loader — src/content.config.ts convention
- withastro/action@v3 for GitHub Actions deployment
- Satori + Sharp for OG image generation (woff fonts from fontsource, not Google Fonts TTF)
- media="print" onload technique for async font loading
- focus-visible outlines for keyboard-only accessibility indicators

**Issues Resolved:**
- Typing animation double-fire after ClientRouter navigation (window.__typingInterval guard)
- Astro 5 API changes (post.id instead of post.slug, standalone render() function)
- expressiveCode must precede mdx in integrations array
- Google Fonts GitHub TTF paths returned 404 (switched to fontsource woff)
- CLS from typing animation container (fixed with min-height)

**Issues Deferred:**
- DNS configuration for patrykgolabek.dev (manual step, outside automation scope)
- Visual/interactive human verification for animations, responsive layouts, keyboard navigation

**Technical Debt Incurred:**
- PG initials as logo placeholder (intentional, to be replaced with proper branding)
