# Milestone v1.16: Claude Code Guide

**Status:** ✅ SHIPPED 2026-03-11
**Phases:** 90-95
**Total Plans:** 17

## Overview

Build a comprehensive multi-chapter Claude Code guide at `/guides/claude-code/` — the definitive zero-to-hero resource covering setup through production mastery and advanced agent workflows. Infrastructure refactoring comes first (7 hardcoded single-guide assumptions to fix with regression gate on FastAPI guide), then SVG diagrams and interactive components to establish visual vocabulary, then 11 MDX chapters split into foundation and advanced tiers, and finally site integration with companion blog post.

## Milestone Summary

**Key Decisions:**
- Removed GUIDE_ROUTES.landing constant — guideLandingUrl() as single source of truth
- Per-guide collection registration pattern (separate collections per guide)
- Dynamic sitemap builder via readdirSync for auto-discovery
- CodeBlock component reuses expressive-code wrapper without source attribution
- Triangular Bezier layout for agentic loop diagram
- Diamond SVG nodes for permission flow explorer (not clip-path)
- Independent data modules for interactive components (no Phase 91 coupling)
- Native install script (curl) as primary installation method; npm deprecated
- HTML entity escaping for ${} references in MDX prose
- Vulnerability scanning framed as capability pattern, not standalone feature
- Blog post "The Context Window Is the Product" frames context management as core skill

**Issues Resolved:**
- Fixed 7 hardcoded single-guide assumptions in infrastructure code
- Resolved React Flow handle clipping by using decorative SVG background instead of clip-path
- SSE transport correctly documented as deprecated with HTTP as recommended replacement

**Issues Deferred:**
- Hub page (/guides/) uses Tailwind dynamic class interpolation for accentColor — should migrate to inline style attributes
- Beauty Index OG image "Cannot find module renderers.mjs" error (pre-existing, unrelated to v1.16)

**Technical Debt Incurred:**
- Hub page Tailwind dynamic class interpolation will break silently if accent hex values change
