EDA techniques divide into **graphical** (visual pattern detection) and **quantitative** (numerical summary) methods. This taxonomy organizes all techniques by the analysis problem they address. See also [Analysis Questions](/eda/reference/analysis-questions/) for guidance on when to apply each category.

## Distributional Analysis

| Technique | Type | Purpose |
|-----------|------|---------|
| [Histogram](/eda/techniques/histogram/) | Graphical | Shape, modality, spread |
| [Box Plot](/eda/techniques/box-plot/) | Graphical | Median, quartiles, outliers |
| [Normal Probability Plot](/eda/techniques/normal-probability-plot/) | Graphical | Normality assessment |
| [Probability Plot](/eda/techniques/probability-plot/) | Graphical | General distributional fit |
| [PPCC Plot](/eda/techniques/ppcc-plot/) | Graphical | Optimal shape parameter |
| [Q-Q Plot](/eda/techniques/qq-plot/) | Graphical | Compare two distributions |
| [Anderson-Darling Test](/eda/quantitative/anderson-darling/) | Quantitative | Formal goodness-of-fit |
| [Chi-Square GOF Test](/eda/quantitative/chi-square-gof/) | Quantitative | Categorical/binned fit |
| [K-S Test](/eda/quantitative/kolmogorov-smirnov/) | Quantitative | Distribution-free GOF |
| [Skewness and Kurtosis](/eda/quantitative/skewness-kurtosis/) | Quantitative | Shape measures |
