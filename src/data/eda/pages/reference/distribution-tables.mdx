These tables provide critical values for common probability distributions used in hypothesis testing and confidence interval construction. For interactive exploration, see the individual [distribution pages](/eda/distributions/).

## Normal Distribution Critical Values

Upper-tail critical values z_alpha where P(Z > z_alpha) = alpha for the standard [Normal Distribution](/eda/distributions/normal/).

| alpha | 0.10 | 0.05 | 0.025 | 0.01 | 0.005 | 0.001 |
|-------|------|------|-------|------|-------|-------|
| z_alpha | 1.282 | 1.645 | 1.960 | 2.326 | 2.576 | 3.090 |

Common two-sided intervals: 90% uses z = 1.645, 95% uses z = 1.960, 99% uses z = 2.576.

## Using These Tables

- **One-tailed test** -- compare test statistic directly to the table value at your chosen alpha
- **Two-tailed test** -- use alpha/2 column (e.g., for 5% two-tailed, use the 0.025 column)
- **Confidence intervals** -- use the appropriate percentile to construct bounds
- For distributions not tabulated here, see [Related Distributions](/eda/reference/related-distributions/) for transformation relationships
